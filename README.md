# cloudflare-ddns-cli

* X_AUTH_KEY: Global API Key.
* ZONE_ID: Can be checked on the web page.
* DNS_RECORD_NAME: The DNS_Record name for your website.
* WEBSITE_URL: URL for your website.
* INTERVAL: Interval between 2 queries for IP address.

## Usage

Log on screen:

```
python ddns.py -p
```

Log to file:

```
python ddns.py
```
